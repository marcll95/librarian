		Get whole ISBN Number in a displayable fashion (see Handbook p. 15)

    access:  public
  









	
        
Parameters:

    

    

          

        string  
        $format  
        Formatstring 1-4 Chars:                each character is a control char:                #1 i or not: use international pre-prefix                #2 i or not: "ISBN" in front or v: incl. version                #3 : or not: insert a ":"                #4 - or not: use - after EAN (ISBN 13 only)                #4 or =: use - between each ISBN part                Example 1:                   '   --' 978-0-385-33941-4                   classic displayable ISBN                Example 2:                   ' v:-' ISBN-13: 978-0385339414                   ISBN-Format used by amazon                Example 3:                   'iv:=' ISBN-13: 978-0-385-33941-4                   full blown: more is more!
